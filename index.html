Segue abaixo algumas informações sobre a utilização da página:
  • O script não contempla pivotamento automático, por isso, altere o sistema linear manualmente;
  • O script verifica se o sistema linear atende os Critérios de Linha e Coluna, caso não passe em nenhum desses,
    aplica o Critério de Sassenfeld(Que permite apenas a utilização do Gauss-Seidel);
  • Para utilizar o modo de Iteração Automática, é necessário definir a Quantidade Máxima de Iterações;
  • Sempre que um dos métodos encontrar um valor de erro menor que o valor informado no campo Erro aceitável,
    irá questionar se você deseja continuar iterando aquele método.
    Se optar por continuar, o script segue rodando até que atinja a quantidade máxima de iterações.
